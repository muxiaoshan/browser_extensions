一、IE扩展程序安装
1、依赖的基础环境.Net Framework v4.0版本，安装脚本将自动检测，如果未安装，则提示“Microsoft.NetFramework 4.0 is not installed. Please install it first.”

2、把deploy放到任意目录，命令行下进入deploy所在目录

3、双击install.bat执行爬虫IE插件的自动安装，windows xp系统下安装时，弹出“运行身份”选择框，在下列用户选项中选择Administrator，并输入密码；

4、IE打开HIS系统进行访问

二、Chrome扩展程序安装
1、打开Chrome扩展程序界面，勾选开发者模式

2、把chrome_close.crx拖动到Chrome界面上

3、在弹出确认框中点击“添加扩展程序”，完成扩展程序安装

4、chrome开始监听web系统发送的关闭当前tab页事件
